# SoundSwitcher

SoundSwitcher is a small KDE/Wayland tray application for switching the default audio output with global keyboard shortcuts.

The entire codebase was created by AI.

## Features

- Qt Widgets settings window
- KDE GlobalAccel shortcuts, suitable for Plasma on Wayland
- PulseAudio-compatible output switching, including PipeWire via `pipewire-pulse`
- System tray menu for quick manual switching
- Settings toggle for per-user autostart
- Desktop and autostart entries generated by CMake

## Dependencies

- CMake 3.24+
- Qt 6 Widgets and DBus development files
- KDE Frameworks 6 GlobalAccel development files
- PulseAudio development files (`libpulse`)
- A PulseAudio-compatible server, such as PulseAudio or PipeWire with `pipewire-pulse`

## Build

```sh
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release -DCMAKE_INSTALL_PREFIX="$HOME/.local"
cmake --build build
cmake --install build
```

The default install creates a per-user autostart entry in `$XDG_CONFIG_HOME/autostart` or `$HOME/.config/autostart`. For a system-wide autostart entry, configure with:

```sh
cmake -S . -B build -DCMAKE_INSTALL_PREFIX=/usr -DSOUNDSWITCHER_AUTOSTART_DIR=/etc/xdg/autostart
```

## Run

```sh
sound-switcher
```

Autostart uses:

```sh
sound-switcher --hidden
```

You can also enable or disable autostart from the settings window.
